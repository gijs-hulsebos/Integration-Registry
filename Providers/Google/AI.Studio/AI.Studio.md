# 🛠 Google AI Studio

> **The fastest way to build with Gemini models.** Google AI Studio is a browser-based IDE for prototyping with generative models, designed to transition smoothly into the Vertex AI enterprise ecosystem.

---

## 💡 Key Capabilities
- **Direct Model Access:** Test Gemini 1.5 Pro, Flash, and Experimental models without setup.
- **Zero-Code Prototyping:** Create system instructions and test prompts in a clean UI before exporting to code.
- **API Key Management:** Simple generation of API keys for integration into external apps.
- **Context Window:** Support for up to 2 million tokens (video, audio, and large codebases).
