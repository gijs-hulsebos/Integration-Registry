# 🕹️ Google Genie (Generative Interactive Environment)

> **The first generative interactive environment.** Genie can take a single image, a sketch, or a text description and transform it into a playable, 2D interactive world with consistent physics and character control.

---

## 💡 Key Capabilities
- **Unsupervised Action Learning:** Learned to control characters by watching unlabelled gameplay videos, without needing action-labels.
- **Image-to-World:** Convert a static AI-generated image (from Imagen or Midjourney) into a functional platformer level.
- **Latent Action Model:** Predicts the next frame based on "user inputs" (Up, Down, Jump) even though it was never explicitly programmed with game mechanics.
